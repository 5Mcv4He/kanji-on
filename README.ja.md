## なぜ「暗記」ではなく「導出」か

中国語母語話者として、あなたは日本人が持たない武器をすでに持っている——**中国語の発音**。

| 既存の知識 | 導出できる音読 |
|:--|:--|
| 拼音声母 `j` | カ行（健·建·見·鍵·剣…） |
| 拼音韻母 `-ou` | 長音ウ（有·由·友·右·油…） |
| 声符「方」 | ホウ（坊·妨·房·放·肪…） |
| 中古音「清」韻 + ing | セイ（清·晴·静·精·情…） |

> これは記憶術ではない。**歴史言語学**だ——日本語漢音は唐代長安音に由来し、現代中国語方言と中古音再構がこの対応関係を逆方向に橋渡しする。

## 五つの鉄則（すぐ使える）

1. **声母が行を決める** — `j` → カ行、`zh` → サ行、`b` → ハ行 … （精度 ~78%）
2. **鼻音が長さを決める** — `-n/-ng` → 長音または撥音（精度 98%+）
3. **声符が音を決める** — 「方」→ ホウ、「包」→ ホウ（精度 ~57-78%）
4. **入声は短い** — 中古入声字 → 短促音節（ツ·ク·キ·チ·フ）
5. **呉漢分離** — 同じ漢字に二つの音読？呉音は濁音保持、漢音は清音化 → カク/ガク 両方説明可能

> 五つの鉄則 ≈ 中学国語知識の日本語版投射。N5 拼音対応を覚えれば、あとは声符と MC 特徴の拡張だけ。
